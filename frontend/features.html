How CommuteWise Works

CommuteWise is an AI-powered multimodal route recommendation system optimized for Chennai. Instead of simply finding the shortest route, it intelligently analyzes multiple travel options using graph algorithms, machine learning, delay prediction, cost estimation, and sustainability metrics to recommend the best route for every commuter.

Key Features

Designed to help commuters make smarter decisions while balancing speed, cost, reliability, and environmental impact.

🧠

AI Delay Prediction

Uses Scikit-Learn regression models trained on historical transit patterns to predict expected travel delays before route selection.

🚇

Multi-Modal Routing

Combines Metro, Bus, Rail, and Walking transfers into a single unified transportation network.

⚡

Fastest Route Detection

Automatically identifies routes with the minimum travel duration and highlights them.

💰

Cheapest Route Detection

Compares estimated expenditures and recommends budget-friendly alternatives.

🌱

Greenest Route

Calculates carbon emissions and promotes environmentally sustainable commuting.

🏆

Smart Ranking System

Ranks routes using weighted scores based on distance, duration, delay, cost and environmental impact.

System Workflow

1

User Input

Source, destination, date and time.

2

Geocoding

Locations converted into coordinates.

3

Graph Search

Dijkstra algorithm generates candidate routes.

4

ML Prediction

Delay prediction using trained regression models.

5

Route Ranking

Routes evaluated using multiple metrics.

6

Results

Best, Fastest, Cheapest and Greenest routes displayed.

Technical Architecture

Frontend

HTML, CSS and JavaScript provide a responsive and intuitive user interface.

Backend

FastAPI powers the route recommendation engine and API communication.

Machine Learning

Scikit-Learn regression models deployed through Pickle files for fast predictions.

Routing Engine

Graph-based Dijkstra implementation with multimodal transport support.

Why CommuteWise?

Unlike traditional route planners that optimize only for distance or travel time, CommuteWise evaluates multiple factors simultaneously including expected delays, transport costs, carbon emissions, and route reliability. This enables commuters to make informed decisions while promoting greener and more efficient urban mobility. Currently optimized for Chennai, the architecture is designed to scale across cities and transit networks.
