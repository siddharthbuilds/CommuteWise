About CommuteWise

A student-built transit intelligence platform for Chennai, combining routing algorithms, machine learning, and real-time data to help commuters travel smarter.

Our Mission

Chennai's public transit network spans metro rail, suburban rail, and an extensive bus system — but navigating across all three to find the right route is rarely straightforward. CommuteWise was built to close that gap. We aggregate the full network into a unified graph, run intelligent pathfinding, and layer in ML-predicted delays and carbon estimates so commuters can make genuinely informed decisions, not just guess.

What We Built

Graph-Based Routing

Dijkstra's algorithm over a transit graph of 1,400+ stops covering Chennai metro, suburban rail, and bus networks with weighted transfer edges.

ML Delay Prediction

A Random Forest model trained on historical delay data, using live weather conditions from Open-Meteo to predict expected delays at query time.

Carbon Estimation

Per-route CO2 estimates based on transport mode and distance, so commuters can weigh environmental impact alongside time and cost.
